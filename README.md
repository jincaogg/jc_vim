#jc_vim
======
    
    jc's vim 

##install
--------
    
    git clone  https://github.com/jincaogg/jc_vim.git  ~/jc_vim

    git submodule init

    git submodule update

    ln -s ~/jc_vim  ~/.vim

    ln -s ~/jc_vim/vimrc ~/.vimrc

or 

    wget https://github.com/jincaogg/jc_vim/archive/master.zip  

    unzip master.zip

    cp master.zip ~/.vim

    ln -s ~/.vim/vimrc ~/.vimrc

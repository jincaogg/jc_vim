#jc_vim
======
    
    jc's vim 

##install
--------
    
    git clone xxx.git ~/jc_vim

    git submodule init

    git submodule update

    ln -s ~/jc_vim  ~/.vim

    ln -s ~/jc_vim/vimrc ~/.vimrc

or 

    wget xxxx.zip 

    unzip xxxx.zip

    cp xxxx ~/.vim

    ln -s ~/.vim/vimrc ~/.vimrc
